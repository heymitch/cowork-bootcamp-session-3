# Cowork Bootcamp — Session 3: Meeting Prep & Email

Your agent becomes your executive assistant. Meeting briefs, email access, and automatic transcripts.

## Install

Add the marketplace, then install the plugin:

```
/plugin marketplace add heymitch/cowork-bootcamp-session-3
/plugin install cowork-bootcamp-session-3@speakeasy-session-3
```

One install — all three skills appear automatically.

## Skills

| Skill | Trigger | What It Does |
|-------|---------|-------------|
| Meeting Prep | "Prep me for my 2pm call" | Build a meeting brief from Gmail, Calendar, and Fireflies |
| Gmail Connector | "Connect Gmail" | Set up Gmail so your agent can read and draft emails |
| Fireflies Connector | "Connect Fireflies" | Set up Fireflies for automatic meeting transcripts |

## Session Order

1. **Gmail Connector** — connect Gmail for email context (5 minutes)
2. **Fireflies Connector** — connect Fireflies for meeting transcripts (5 minutes)
3. **Meeting Prep** — prep for your next meeting using all connected sources
